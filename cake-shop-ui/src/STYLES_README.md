# Структура стилей проекта CakeFactory

## Обзор

Проект использует единую, централизованную систему стилей для обеспечения консистентности дизайна по всему приложению.

## Структура файлов

### 1. `index.css` (глобальные стили - 28 строк)
Содержит:
- **CSS переменные** (--bg-main, --accent, --text-main и т.д.)
- **Глобальные стили** для body, *, a
- Является базовым файлом, который загружается первым

**Назначение**: Определение цветовой схемы и глобальных стилей сайта.

### 2. `App.css` (все компонентные стили - 1514 строк)
Содержит полный набор стилей для всех компонентов и страниц:
- Navbar
- Hero секции
- Product cards
- Form elements
- Account & Admin pages
- Custom Cake builder
- Responsive стили

**Назначение**: Все компонентные и страничные стили в одном файле для легкого управления.

## Цветовая схема

Все цвета определены как CSS переменные в `index.css`:

```css
--bg-main: #fdeff4       /* Основной фон */
--bg-section: #fff7f9    /* Фон секций */
--bg-card: #ffffff       /* Фон карточек */
--accent: #f9a8c6        /* Основной акцент */
--accent-dark: #f284b5   /* Темный акцент */
--text-main: #2a1727     /* Основной текст */
--text-muted: #9b7c90    /* Приглушенный текст */
--badge-bg: #fde1ea      /* Фон бейджей */
--shadow-soft: 0 18px 40px rgba(0, 0, 0, 0.08)  /* Мягкая тень */
--radius-large: 28px     /* Большой border-radius */
```

## Как использовать

### Импорт стилей

В `index.js` стили импортируются в правильном порядке:

```javascript
import "./index.css";  // Глобальные переменные и стили
import "./App.css";    // Все компонентные стили
```

### Использование цветовых переменных

```css
.my-element {
  background: var(--accent);
  color: var(--text-main);
  box-shadow: var(--shadow-soft);
  border-radius: var(--radius-large);
}
```

## Принципы организации

1. **DRY (Don't Repeat Yourself)** - нет дублирования стилей
2. **Централизованное управление** - все стили в двух файлах
3. **Единая цветовая схема** - через CSS переменные
4. **Консистентность** - одинаковые стили на всех страницах
5. **Responsive Design** - мобильные стили встроены в App.css

## Responsive breakpoints

Проект использует следующие breakpoints:
- `max-width: 960px` - планшеты
- `max-width: 900px` - средние экраны
- `max-width: 800px` - маленькие планшеты
- `max-width: 640px` - мобильные устройства

## Изменение дизайна

### Смена цветовой схемы

Измените значения переменных в `index.css`:

```css
--bg-main: #новый_цвет;
--accent: #новый_цвет;
/* и т.д. */
```

Изменения автоматически применятся ко всему сайту.

### Добавление новых стилей

1. Добавьте новые стили в `App.css`
2. Используйте существующие CSS переменные где возможно
3. Следуйте существующей структуре и названиям классов
