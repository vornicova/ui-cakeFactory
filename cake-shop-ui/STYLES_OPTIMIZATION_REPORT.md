### 📱 Responsive стили

Проект поддерживает следующие breakpoints:
- **960px** - планшеты
- **900px** - средние экраны  
- **800px** - маленькие планшеты
- **640px** - мобильные устройства

Все @media запросы находятся в `App.css` и применяют одинаковые правила ко всем компонентам.

## Преимущества

✅ **Консистентность** - все стили на всех страницах одинаковые
✅ **Единое управление** - цвета меняются в одном месте
✅ **Меньше кода** - нет дублирования
✅ **Легче поддержка** - логичная структура
✅ **Лучшая производительность** - меньше CSS для парсинга

## Файлы, которые НЕ импортируют CSS отдельно

Все компоненты используют глобально импортированные стили:

```javascript
// src/index.js
import "./index.css";  // Переменные и базовые стили
import "./App.css";    // Все компонентные стили
```

Никакие компоненты (JSX файлы) не должны иметь собственные CSS импорты.

## Что изменилось для разработчиков

Если вы хотите:

### Изменить цвет
Отредактируйте значение в `src/index.css`:
```css
--accent: #новый_цвет;
```
Изменение применится ко всему сайту автоматически.

### Добавить новый стиль
Добавьте его в `src/App.css`:
```css
.my-new-component {
  background: var(--bg-card);
  color: var(--text-main);
  /* ... */
}
```

### Добавить responsive версию
Добавьте @media правило в конец `src/App.css`:
```css
@media (max-width: 640px) {
  .my-new-component {
    /* мобильные стили */
  }
}
```
